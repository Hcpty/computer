# 数据结构和算法

### File System
- Storing
  - Block
- Indexing
  - Inode
- Sorting (Similar to Relational Database System)

### Relational Database System
- Storing
  - Table
- Indexing
  - Hash Table
  - Search Tree
  - Skip List
- Sorting
  - In Memory Sorting Algorithm
  - On Disk Sorting Algorithm

### Key-Value Database System
- Storing (Vary from one to another)
- Indexing
  - Hash Table
- Sorting (Similar to Relational Database System)

### Queueing System
- Priority Queue
